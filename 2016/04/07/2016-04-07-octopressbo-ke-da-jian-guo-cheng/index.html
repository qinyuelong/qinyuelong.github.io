Octopress 精益修改

日期格式

date_format: “ordinal” #默认日期显示方式
请把其中的 “ordinal” 改为 “%Y 年%-m 月%-d 日”，并保存

文章链接形式

permalink: /blog/:title/ #实现第二种

分类目录前缀

添加下面一行代码并保存
category_title_prefix: “分类：” # 修改分类前缀

理论上可以添加到任何位置，但为了显示直观，建议加到 category_dir: blog/categories 下一行

「继续阅读」按钮

excerpt_link: 「继续阅读」

修改到这一步，你肯定很想看看效果，现在执行如下命令，重新生成并部署页面

rake generate
rake deploy

遇到的问题

修改 Rakefile 的 markdown 后缀名后引起不能部署(不要修改)

起因：修改后并推送到 source 后 不能部署
解决:

  1. 本地代码回滚 git reset –hard <要rollback到的commit>
  2. 远程回滚 git push source HEAD –force
